<h1 align="center"> Clima360</h1>


<h3>Una página de clima muestra pronósticos, temperaturas y condiciones climáticas actuales de diferentes ubicaciones geográficas de manera visual. a los usuarios explorar una variedad de títulos cinematográficos y encontrar fácilmente sus favoritos.</h3>
<h3>A weather page displays forecasts, temperatures, and current weather conditions visually across various geographic locations, aiding users in exploration.</h3>

- 📝 git clone **https://github.com/EmanuelRigo/Clima360.git**

- 📄 npm **install**

- ⚡ npm **run dev**
